Que es PIP?

Es un sistema que incorpora Python para administrar paquetes e instalarlos. Una de sus ventajas es la facilidad de uso.

Como instalar PIP

En primer lugar comprobaremos si tenemos PIP instalado.

Para ello ejecutamos en nuestra terminal o consola de comandos.


						pip --version

Como instalar PIP

Linux

Dependiendo de la versión de Python que vayamos a utilizar, instalaremos una u otra versión de PIP.


						sudo apt-get install python-pip
						sudo apt-get install python3-pip

Como instalar PIP

Windows

En principio, al instalar Python siguiendo los pasos y añadir Python al path, no tendriamos problema con pip y no necesitariamos instalarlo. Pero hay una solución por si surgen problemas.

Tendremos que descargar un script desde este link.

pyTelegramBotAPI

Es un módulo de Python que nos va a permitir interactuar con la api de Telegram.

Podeis encontrar una gran cantidad de documentacion del mismo en su github, ya que no podremos tratarlo todo en este taller.

Instalacion del modulo

Usaremos PIP para instalar el modulo. 


						pip install pyTelegramBotAPI
						python -m pip install pyTelegramBotAPI

Tras realizar la instalacion del modulo ya podremos importarlo desde cualquier script python.

Tratando comandos

Vamos a explicar este pequeño trozo de código.
En primer lugar, seguido de un @ vemos un decorador de python. 


						@bot.message_handler(commands=['prueba'])
						def bienvenida(message):
							bot.reply_to(message, "Hola, esto funciona")

Tratando comandos

Usaremos el decorador para filtrar el caso en el que el bot ejecutará esa función.
En este caso, solo se ejecutará si ponemos /prueba. 


						@bot.message_handler(commands=['prueba'])
						def bienvenida(message):
							bot.reply_to(message, "Hola, esto funciona")

Tratando comandos

Este pequeño código devuelve el mensaje que recibimos.
En este caso, todos los mensajes pasan a través del decorador. 


						@bot.message_handler(func=lambda m: True)
						def echo_all(message):
							bot.reply_to(message, message.text)

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );
