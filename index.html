Ejecución de un bot

Un bot no es un programa que ejecutemos y finalice, realizando una acción concreta.
Es un programa que responde a llamadas que no sabemos cuando se van a producir, por tanto debe tener un sistema de espera síncrona o asíncrona.

Ejecución de un bot

Usaremos polling, que explicado de una manera sencilla, se dedica a preguntar repetidas veces al programa si hay algo que tratar, hasta que suceda algún evento y pueda actuar.

Por tanto debemos añadir esto al final del programa. 


						bot.polling()

						m = message.from_user.first_name
						if message.from_user.last_name is not None:

						bot.reply_to(message,m )

Cread un bot que descargue o guarde el id de las fotos que le envian y que envie solo a tu usuario una de esas fotos (en orden) cuando lanzas el comando /get

Ejemplo:
User1 manda foto1
User1 manda foto2
Admin /get enviando foto1
Admin /get enviando foto2

Solución propuesta


from queue import Queue

photos = Queue()

@bot.message_handler(content_types=['voice'])
def store_photo(message):
	photos.put(message.photo.file_id)

@bot.message_handler(commands=['get'])
def get_photo(message):
	photo_id = photos.get()
	bot.send_photo(message.chat.id , photo_id)
